# CRM Console Application

A **console-based Customer Relationship Management (CRM) system** developed using **C++**.

This project represents an **improved and extended version** of a previous CRM system that was originally developed using **Windows Forms and C#**.

The goal of this version was to rebuild the system in a **console environment** while improving the internal structure and focusing more on **core system logic and object-oriented design**.

---

# Project Purpose

This project was built as part of a **learning and development process** to better understand how larger applications are structured.

The main objectives of the project include:

* Implementing a **login and authentication system**
* Managing **system users**
* Managing **client data**
* Implementing **user permissions**
* Organizing program logic using **Object-Oriented Programming (OOP)**
* Working with **file-based data storage**

The development of this project was **incremental**, meaning the system was improved and expanded step by step while learning new concepts.

---

# Features

The system includes several core CRM functionalities:

### Authentication

* User login system
* Basic access control

### Client Management

* Add new clients
* Update client information
* Delete clients
* Search for clients
* Display clients list

### User Management

* Add system users
* Update users
* Delete users
* Manage permissions
